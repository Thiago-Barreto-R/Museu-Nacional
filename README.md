## Projeto Museu Nacional

O objetivo desse projeto foi para colocar em prática os conhecimentos da semântica do **HTML 5**, **CSS 3** com *animações* e com o uso do Pré-processador **SASS**.

### Tecnologias utilizadas

* HTML 5
* CSS 3
* SASS  
  * SCSS
